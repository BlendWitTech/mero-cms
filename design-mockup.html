Questions

Everything you'd ask before buying.

Can't find the answer? Talk to a human →

Is Mero CMS a subscription?
No. Mero CMS is a one-time license per production deployment. You own the running code. Lifetime updates for the current major version, plus 12 months of support included.
Can we build our own theme?
Yes. Every theme is a Next.js project that declares sections and variants in theme.json. The admin adapts automatically — no extra CMS work to add new sections.
Does it work with our existing stack?
Mero CMS is a headless-ish Node + Postgres backend with REST endpoints. Themes are Next.js apps consuming those endpoints. API keys let mobile, native, or any other frontend pull the same content.
What about performance?
The theme is statically generated with ISR. The backend ships with helmet, rate limits, HMAC-signed webhooks, and a 2 MB JSON body cap. Private media goes through signed URLs.
How does upgrading tiers work?
It's a license swap, not a reinstall. Capabilities gate via the matrix in code, so the right features turn on the moment your tier updates. Downgrades are supported — features hide, content stays.
